🤖 Ravis

Assistente virtual desktop inspirado no JARVIS

🚀 Sobre

O Ravis é um assistente virtual com foco em IA em tempo real, automação do sistema e uma interface futurista estilo HUD.

Combina chat inteligente, voz, visão computacional e memória persistente para criar uma experiência fluida e interativa.

✨ Funcionalidades
💬 Chat com IA (streaming)
🧠 Memória inteligente
🔊 Respostas por voz (TTS)
👁️ Análise de tela (PrintScreen)
🌐 Pesquisa web integrada
⚙️ Controle do sistema
🎯 Atalhos globais
⚙️ Tecnologias
Backend: Python, FastAPI, WebSocket
Frontend: HTML, CSS, JavaScript, Tailwind
IA: Groq, Gemini
Sistema: mss, Pillow, psutil, pynput
▶️ Como rodar
pip install -r requirements.txt
python main.py

Acesse:

http://localhost:8000
🔧 Configuração

Crie um .env:

GROQ_API_KEY=
GEMINI_API_KEY=
TAVILY_API_KEY=
SERPER_API_KEY=
SEARXNG_URL=http://localhost:8080
🎮 Atalho
PrintScreen → ativa análise de tela
